/ Nyocerifier /



In the early 2000s Kyocera made a series of cameras under the name Finecam.
These cameras are known for having a very unique look to them and this website attempts to emulate that look.
You can upload an image here and it will be processed to look like it came from one of these cameras.

This website is Free Software and the source code can be found here.
